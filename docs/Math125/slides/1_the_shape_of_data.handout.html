5 Fetching and viewing data with Jupyter – illustration

  • Make a Jupyter notebook using URL colab.new
  • Find OpenIntro Earthquake data CSV link on openintro.org/data
  • Fetch data in a code cell:
!wget [url to .csv file]
  • Import pandas:
import pandas as pd
  • Read and display data:
df = pd.read_csv("earthquakes.csv")
df.head()
  • Analysis:
df.describe()
df.corr(numeric_only=True)

6 Data into Jupyter – practice

  1. Find the openintro data repository
  2. Locate “acs12” data source. Copy link location to the .csv format.
  3. Create a new jupyter notebook on colab, using URL shortcut “colab.new.”
  4. If you’re blind, change the Colab option in Tools, Settings, Editor to turn off the option called “Show context-powered code completions.”
  5. Name your notebook.
  6. Use wget to download the dataset at that url.
  7. Use pandas to read the dataframe.
  8. Display the dataframe.
  9. Use describe() and corr() for analysis.
